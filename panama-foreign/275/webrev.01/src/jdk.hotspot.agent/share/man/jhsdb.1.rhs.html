104 .SH DESCRIPTION
105 .PP
106 You can use the \f[CB]jhsdb\f[R] tool to attach to a Java process or to
107 launch a postmortem debugger to analyze the content of a core\-dump from
108 a crashed Java Virtual Machine (JVM).
109 This command is experimental and unsupported.
110 .PP
111 \f[B]Note:\f[R]
112 .PP
113 Attaching the \f[CB]jhsdb\f[R] tool to a live process will cause the
114 process to hang and the process will probably crash when the debugger
115 detaches.
116 .PP
117 The \f[CB]jhsdb\f[R] tool can be launched in any one of the following
118 modes:
119 .TP
120 .B \f[CB]jhsdb\ clhsdb\f[R]
121 Starts the interactive command\-line debugger.
122 .RS
123 .RE
124 .TP
125 .B \f[CB]jhsdb\ hsdb\f[R]
126 Starts the interactive GUI debugger.
127 .RS
128 .RE
129 .TP
130 .B \f[CB]jhsdb\ debugd\f[R]
131 Starts the remote debug server.
132 .RS
133 .RE
134 .TP

136 Prints stack and locks information.
137 .RS
138 .RE
