 26 .SH NAME
 27 .PP
 28 jmap \- print details of a specified process
 29 .SH SYNOPSIS
 30 .PP
 31 \f[B]Note:\f[R] This command is experimental and unsupported.
 32 .PP
 33 \f[CB]jmap\f[R] [\f[I]options\f[R]] \f[I]pid\f[R]
 34 .TP
 35 .B \f[I]options\f[R]
 36 This represents the \f[CB]jmap\f[R] command\-line options.
 37 See \f[B]Options for the jmap Command\f[R].
 38 .RS
 39 .RE
 40 .TP
 41 .B \f[I]pid\f[R]
 42 The process ID for which the information specified by the
 43 \f[I]options\f[R] is to be printed.
 44 The process must be a Java process.
 45 To get a list of Java processes running on a machine, use either the
 46 \f[CB]ps\f[R] command or, if the JVM processes are not running in a
 47 separate docker instance, the \f[B]jps\f[R] command.
 48 .RS
 49 .RE
 50 .SH DESCRIPTION
 51 .PP
 52 The \f[CB]jmap\f[R] command prints details of a specified running process.
 53 .PP
 54 \f[B]Note:\f[R]
 55 .PP
 56 This command is unsupported and might not be available in future
 57 releases of the JDK.
 58 On Windows Systems where the \f[CB]dbgeng.dll\f[R] file isn\[aq]t present,
 59 the Debugging Tools for Windows must be installed to make these tools
 60 work.
 61 The \f[CB]PATH\f[R] environment variable should contain the location of
 62 the \f[CB]jvm.dll\f[R] file that\[aq]s used by the target process or the
 63 location from which the core dump file was produced.
 64 .SH OPTIONS FOR THE JMAP COMMAND
 65 .TP
 66 .B \f[CB]\-clstats\f[R] \f[I]pid\f[R]
 67 Connects to a running process and prints class loader statistics of Java
 68 heap.
 69 .RS
 70 .RE
 71 .TP
 72 .B \f[CB]\-finalizerinfo\f[R] \f[I]pid\f[R]
 73 Connects to a running process and prints information on objects awaiting
 74 finalization.
 75 .RS
 76 .RE
 77 .TP
 78 .B \f[CB]\-histo\f[R][\f[CB]:live\f[R]] \f[I]pid\f[R]
 79 Connects to a running process and prints a histogram of the Java object
 80 heap.
 81 If the \f[CB]live\f[R] suboption is specified, it then counts only live
 82 objects.
 83 .RS
 84 .RE
 85 .TP
 86 .B \f[CB]\-dump:\f[R]\f[I]dump_options\f[R] \f[I]pid\f[R]
 87 Connects to a running process and dumps the Java heap.
 88 The \f[I]dump_options\f[R] include:
 89 .RS
 90 .IP \[bu] 2
 91 \f[CB]live\f[R] \-\-\- When specified, dumps only the live objects; if not
 92 specified, then dumps all objects in the heap.
 93 .IP \[bu] 2
 94 \f[CB]format=b\f[R] \-\-\- Dumps the Java heap in \f[CB]hprof\f[R] binary
 95 format
 96 .IP \[bu] 2
 97 \f[CB]file=\f[R]\f[I]filename\f[R] \-\-\- Dumps the heap to
 98 \f[I]filename\f[R]
 99 .PP
100 Example: \f[CB]jmap\ \-dump:live,format=b,file=heap.bin\f[R] \f[I]pid\f[R]
101 .RE
