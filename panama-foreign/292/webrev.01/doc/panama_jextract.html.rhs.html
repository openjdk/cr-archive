173 <h1 id="using-panama-foreign-jextract-jdk">Using Panama "foreign-jextract" JDK</h1>
174 <p>You can build "foreign-jextract" branch of panama repo <a href="https://github.com/openjdk/panama-foreign">https://github.com/openjdk/panama-foreign</a></p>
175 <p>Using foreign function call in Java involves the following two steps:</p>
176 <ol type="1">
177 <li>Use <strong>jextract</strong> tool to generate java interface for your C header file(s)</li>
178 <li>Invoke C functions via the jextracted Java interface</li>
179 </ol>
