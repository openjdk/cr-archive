 682 // Here are all the intrinsics known to the runtime and the CI.
 683 // Each intrinsic consists of a public enum name (like _hashCode),
 684 // followed by a specification of its klass, name, and signature:
 685 //    template(<id>,  <klass>,  <name>, <sig>, <FCODE>)
 686 //
 687 // If you add an intrinsic here, you must also define its name
 688 // and signature as members of the VM symbols.  The VM symbols for
 689 // the intrinsic name and signature may be defined above.
 690 //
 691 // Because the VM_SYMBOLS_DO macro makes reference to VM_INTRINSICS_DO,
 692 // you can also define an intrinsic's name and/or signature locally to the
 693 // intrinsic, if this makes sense.  (It often does make sense.)
 694 //
 695 // For example:
 696 //    do_intrinsic(_foo,  java_lang_Object,  foo_name, foo_signature, F_xx)
 697 //     do_name(     foo_name, "foo")
